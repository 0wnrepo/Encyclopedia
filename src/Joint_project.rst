================================================================================
Joint Project
================================================================================

A **joint project** is a project that can only be accomplished by the
participants working together toward a common goal.[1] For example, rowing a
boat.

Joint projects differ in size.[1]

Minimal Joint project
================================================================================

Adjacency pairs form units in which people achieve certain goals together,
certain joint purposes.[1]

Adjacency pairs are minimal joint projects.[1]
